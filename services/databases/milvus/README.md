# Milvus

Milvus is the vector database used by AABS to store and search embeddings generated by the embedding service.

## Ports

| Port | Description |
|--------|-------------|
| 19530 | Milvus gRPC API |
| 9091 | Health endpoint |
| 9000 | MinIO API |
| 9001 | MinIO Console |

## Health Check

Verify that Milvus is running:

bash curl http://localhost:9091/healthz 

Expected response:

text OK 

## View Logs

bash docker logs -f aabs-milvus 

## View Resource Usage

bash docker stats aabs-milvus 

## Connect From Go

Milvus endpoint:

text localhost:19530 

From another container in the same Docker network:

text milvus:19530 

## Data Persistence

Milvus data is stored in:

text ./data/milvus 

Metadata is stored in:

text ./data/etcd 

Object storage is provided by MinIO:

text ./data/minio
